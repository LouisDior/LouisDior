Servidor LiteSpeed


LiteSpeed ​​fue el primero en introducir el soporte de servidor web HTTP / 3 de nivel de producción. Pero no nos detuvimos ahí. A medida que se sigue definiendo el nuevo protocolo, estamos presentes en cada paso, manteniendo nuestra implementación actualizada y de vanguardia. Todos nuestros productos de servidor en general cuentan con soporte HTTP / 3.


LiteSpeed ​​fue diseñado específicamente para aprovechar los beneficios de HTTP / 3 y QUIC. Como era de esperar, QUIC.cloud ofrece contenido a través de HTTP / 3 a los visitantes, pero también puede comunicarse con el backend de un sitio a través de HTTP / 3 y QUIC, lo que crea una conexión completa con HTTP / 3.


Caracteristicas Principales


  • Duplique la capacidad máxima de sus servidores Apache actuales con la arquitectura optimizada basada en eventos de LiteSpeed ​​Web Server, capaz de manejar miles de clientes simultáneos con un consumo mínimo de memoria y uso de CPU.

  • Proteja sus servidores con las reglas mod_security que ya conoce y al mismo tiempo aproveche una serie de funciones anti-DDoS integradas, como el ancho de banda y la limitación de la conexión.

  • Ahorre capital reduciendo la cantidad de servidores necesarios para respaldar su creciente negocio de alojamiento o aplicación en línea.

  • Reduzca la complejidad al eliminar la necesidad de un proxy inverso HTTPS o capas adicionales de almacenamiento en caché de terceros.
